# Bank Loan Analytics Dashboard (Power BI)

## Objective
The objective of this project is to analyze bank loan data and provide insights into loan applications, funded amounts, loan performance, and borrower characteristics using an interactive Power BI dashboard.

This dashboard helps in understanding loan distribution, monitoring key KPIs, and identifying good and bad loans for better decision-making.

---

# Dashboard Overview

The Power BI dashboard contains **three main report pages**:

## 1️⃣ BANK LOAN REPORT | SUMMARY
This page shows high-level **Key Performance Indicators (KPIs)** such as:

- Total Loan Applications  
- Month-To-Date (MTD) Loan Applications  
- Total Funded Amount  
- Total Amount Received  
- Average Interest Rate  
- Average Debt-to-Income Ratio (DTI)  
- Good Loans vs Bad Loans  
- Loan Status Distribution  

These KPIs help in quickly understanding the overall loan portfolio performance.

---

## 2️⃣ BANK LOAN REPORT | OVERVIEW
This page provides **loan distribution analysis** across different dimensions:

- Loan Applications by Month  
- Loan Applications by State  
- Loan Term Analysis  
- Employee Length Distribution  
- Loan Purpose Analysis  
- Home Ownership Distribution  

This helps identify **trends and patterns in loan applications**.

---

## 3️⃣ BANK LOAN REPORT | DETAILS
This page provides a **detailed tabular view of loan records**.

It includes information such as:

- Loan ID  
- Purpose of Loan  
- Home Ownership  
- Loan Grade & Sub Grade  
- Issue Date  
- Total Funded Amount  
- Installment Amount  
- Total Amount Received  
- Interest Rate  

Users can apply filters such as **State, Grade, and Good vs Bad Loan** to analyze specific records.
